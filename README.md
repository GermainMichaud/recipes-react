## Getting started

### Clone repository

```bash
git clone git@github.com:GermainMichaud/recipes-react.git
```

### Install dependencies

```bash
cd ./devtalk

npm i
# or
yarn
```

### Run project (development)

```bash
npm run dev
# or
yarn dev
```

### Build project

It will build application in folder `dist`

```bash
npm run build
# or
yarn build
```
